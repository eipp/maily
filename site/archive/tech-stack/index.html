    Maily Technology Stack¶

    Last updated: March 4, 2025

    This document provides a comprehensive overview of the technology stack used in the Maily platform. It serves as a reference for developers, architects, and stakeholders to understand the technologies, frameworks, libraries, and tools that power our email marketing platform.

    Frontend Technologies¶

    Core Framework¶

    • Next.js: v14.2.0 - React framework with App Router architecture
    • React: v18.3.0 - UI component library
    • TypeScript: v5.4.2 - Typed JavaScript

    UI & Styling¶

    • Tailwind CSS: v4.1.0 - Utility-first CSS framework
    • shadcn/ui: v3.0.0 - Component library built on Radix UI
    • Radix UI: v3.0.0 - Headless UI components
    • Framer Motion: v11.0.0 - Animation library
    • next-themes: v1.0.0 - Theme management for Next.js

    State Management¶

    • TanStack Query (React Query): v6.0.0 - Server state management
    • Zustand: v5.0.0 - Client state management
    • React Hook Form: v8.0.0 - Form state management
    • Zod: v3.22.0 - Schema validation

    Data Visualization¶

    • Recharts: v2.10.0 - React charting library
    • react-grid-layout: v2.0.0 - Grid layout system for dashboards
    • visx: v3.8.0 - Low-level visualization components

    Internationalization¶

    • next-intl: v3.5.0 - Internationalization for Next.js
    • Formatjs: v2.0.0 - Internationalization framework
    • date-fns: v3.0.0 - Date utility library with i18n support

    Testing¶

    • Vitest: v1.5.0 - Unit testing framework
    • Testing Library: v15.0.0 - Component testing utilities
    • Playwright: v2.0.0 - End-to-end testing
    • MSW: v2.0.0 - API mocking

    Backend Technologies¶

    API Framework¶

    • Express: v4.18.2 - Node.js web framework
    • Fastify: v5.0.0 - High-performance Node.js framework
    • GraphQL: v16.8.0 - API query language
    • Apollo Server: v4.9.0 - GraphQL server
    • Pothos: v4.0.0 - Code-first GraphQL schema builder

    Authentication & Authorization¶

    • NextAuth.js: v5.0.0 - Authentication for Next.js
    • Auth0: - Identity platform
    • JWT: v9.0.0 - JSON Web Tokens
    • CASL: v7.0.0 - Authorization library

    Database¶

    • PostgreSQL: v16.0 - Relational database
    • PgBouncer: v1.22.0 - Connection pooling for PostgreSQL
    • Prisma: v5.10.0 - ORM for TypeScript
    • Redis: v7.2.0 - In-memory data store
    • MongoDB: v7.0.0 - Document database (specific features)

    Messaging & Event Streaming¶

    • Kafka: v3.6.0 - Event streaming platform
    • Redis Streams: v7.2.0 - Streaming data structure
    • WebSockets: - Real-time communication

    AI & Machine Learning¶

    Core AI¶

    • Anthropic Claude 3: - Large language model (Opus, Sonnet, Haiku)
    • LiteLLM: v1.0.0 - LLM router and orchestration
    • LangChain: v0.2.0 - LLM framework
    • Stability AI: - Image generation

    ML Ops¶

    • Weights & Biases: - ML experiment tracking
    • DVC: v3.30.0 - ML versioning
    • MLflow: v2.10.0 - ML lifecycle management

    AI Observability¶

    • Langfuse: v2.0.0 - LLM observability
    • Arize AI: - ML monitoring
    • Helicone: v2.0.0 - LLM cost management

    ETL & Data Pipeline¶

    • Apache Airflow: v2.8.0 - Workflow management
    • dbt: v1.6.0 - Data transformation
    • Feast: v0.35.0 - Feature store

    DevOps & Infrastructure¶

    Containerization & Orchestration¶

    • Docker: v25.0.0 - Containerization
    • Kubernetes: v1.30.0 - Container orchestration
    • Helm: v3.14.0 - Kubernetes package manager

    CI/CD¶

    • GitHub Actions: - CI/CD platform
    • ArgoCD: v2.10.0 - GitOps CD
    • Turborepo: v2.0.0 - Monorepo build system

    Monitoring & Observability¶

    • OpenTelemetry: v1.22.0 - Observability framework
    • Prometheus: v2.50.0 - Metrics collection
    • Grafana: v10.3.0 - Metrics visualization
    • Tempo: v2.3.0 - Distributed tracing
    • Loki: v2.9.0 - Log aggregation

    Infrastructure as Code¶

    • Terraform: v1.7.0 - Infrastructure provisioning
    • Pulumi: v3.100.0 - Infrastructure as code

    Cloud Providers & Services¶

    • AWS: Primary cloud provider
    • EKS: Kubernetes service
    • RDS: Database service
    • S3: Object storage
    • CloudFront: CDN
    • Lambda: Serverless compute
    • EventBridge: Event bus
    • SQS: Message queuing
    • Route53: DNS service
    • WAF: Web application firewall
    • Global Accelerator: Network acceleration
    • Cloudflare: Edge services
    • Workers: Edge computing
    • KV: Edge key-value store
    • D1: Edge SQL database
    • R2: Object storage

    Security & Compliance¶

    Security Tools¶

    • Trivy: v0.47.0 - Container security scanning
    • Snyk: - Vulnerability scanning
    • OWASP ZAP: v2.14.0 - Web application security scanner
    • SonarQube: v10.3.0 - Code quality and security

    Secrets Management¶

    • HashiCorp Vault: v1.15.0 - Secrets management
    • AWS Secrets Manager: - Cloud secrets management

    Policy & Compliance¶

    • Open Policy Agent (OPA): v0.60.0 - Policy enforcement
    • Kyverno: v1.11.0 - Kubernetes policy management

    Development Tools¶

    Code Quality¶

    • ESLint: v9.0.0 - JavaScript linting
    • Prettier: v3.1.0 - Code formatting
    • TypeScript: v5.4.2 - Type checking
    • husky: v9.0.0 - Git hooks

    Documentation¶

    • Docusaurus: v3.1.0 - Documentation site generator
    • Swagger/OpenAPI: v3.0.0 - API documentation
    • Storybook: v8.0.0 - Component documentation

    Package Management¶

    • pnpm: v9.0.0 - Fast, disk space efficient package manager
    • Changesets: v2.0.0 - Versioning and changelog management

    Version Control¶

    • Git: v2.43.0 - Version control system
    • GitHub: - Repository hosting and collaboration

    Project Management¶

    • Jira: - Issue tracking
    • Confluence: - Documentation
    • Slack: - Team communication

    Analytics & Reporting¶

    • Mixpanel: - Product analytics
    • PostHog: v1.60.0 - Product analytics and feature flags
    • Cube.js: v0.35.0 - Analytics API
    • Metabase: v0.48.0 - Business intelligence

    Email Infrastructure¶

    • Amazon SES: - Email sending service
    • MJML: v4.15.0 - Responsive email framework
    • Mailparser: v3.6.0 - Email parsing library
    • nodemailer: v6.9.0 - Email sending library

    Standards & Protocols¶

    • OAuth 2.0: - Authorization framework
    • OpenID Connect: - Identity layer
    • SMTP: - Email transmission
    • REST: - API architecture
    • GraphQL: - API query language
    • WebSockets: - Real-time communication

    Integration Services¶

    • Stripe: - Payment processing
    • Segment: - Customer data platform
    • Zapier: - Workflow automation
    • Twilio: - Communication APIs
    • Intercom: - Customer messaging platform

    Mobile Support¶

    • React Native: v0.73.0 - Cross-platform mobile development
    • Expo: v50.0.0 - React Native framework
    • Progressive Web App (PWA): - Web-based mobile experience

    Conclusion¶

    This technology stack represents the current state of the Maily platform as of March 2025. The stack is continuously evaluated and updated to ensure we're using the most appropriate, secure, and performant technologies for our needs.

    For questions about specific technologies or implementation details, please refer to the respective documentation or contact the technical lead responsible for that area of the platform.
